## Лабораторная работа "Обучающая, проверочная и тестовая выборки. Переобучение НС"

## Структура репозитория

- `laba 2 ДЗ_Lite.ipynb` –  ноутбук для задания Lite: построение и сравнение 9 моделей нейронной сети для классификации одежды из набора Fashion MNIST. В работе варьируются архитектуры (количество нейронов, функции активации)
- `laba 2 ДЗ_Pro.ipynb` – ноутбук для задания Pro: классификация вин с использованием датасета из библиотеки sklearn. Выполняется загрузка, предобработка данных и строгое разделение на обучающую, проверочную и тестовую выборки. Проводится серия экспериментов с моделью для достижения точности на тестовой выборке выше 94%.
- `laba 2 ДЗ_Ultra_Pro.ipynb` – ноутбук для задания Ultra Pro: подготовка и обучение нейронной сети для бинарной классификации изображений пассажиров автобуса (вход/выход)
- `Pytorch laba 2.ДЗ Lite.ipynb` – ноутбук для задания Lite: построение и сравнение 9 моделей нейронной сети для классификации одежды из набора Fashion MNIST. В работе варьируются архитектуры (количество нейронов, функции активации). Выполнено с использованием Pytorch

## Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
